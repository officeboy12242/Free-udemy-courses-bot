/enroll_setup
📝 Setup your Udemy credentials (one-time)
   Asks for: access_token & client_id from browser cookies
   Takes: 2 minutes
   Result: Ready to scrape courses

/set_token <token>
🔑 Set access_token directly via command
   Usage: /set_token eyJhbGc...
   Alternative to interactive setup

/set_client_id <id>
🔑 Set client_id directly via command
   Usage: /set_client_id 6U...
   Alternative to interactive setup

/enroll
📚 Scrape 600+ free Udemy courses
   From: 8 coupon sites (DiscUdemy, Real.Discount, etc.)
   Time: 1-2 minutes to scrape
   Result: Course counts per site
   Requirement: Must run /enroll_setup first

/enroll_status
📊 View your scrape results & statistics
   Shows: Courses found, breakdown per site, samples
   Tracks: Your lifetime stats

/myprofile
👤 View your profile & manage credentials
   Shows: Setup status, total scrapes, stats
   Options: Update credentials, delete all data
